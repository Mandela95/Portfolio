# Mohamed El-Seady Portfolio

live-server: https://mandela95.github.io/Portfolio/

A modern, fully responsive portfolio website showcasing my work as a Front-End Developer and Mobile Developer with 3+ years of hands-on experience in web and mobile application development.

# Key Features ✨

📱 Fully Responsive — Mobile-first design optimized for all device sizes

🌓 Dark/Light Theme Toggle — Seamless theme switching with localStorage persistence

🌐 Bilingual Support — Full Arabic & English localization with RTL support

🔗 Social Integration — Brand-colored hover effects for all platforms

⚡ Performance Optimized — Smooth animations, lazy loading, and fast load times

🧭 Timeline Summary — Career journey from self-taught development to professional software engineering roles worldwide.

🎯 Project Showcase — 9 featured projects with filtering by technology

💬 Testimonials — Featured LinkedIn recommendation from a colleague, adding professional credibility and showing reliability, ownership, collaboration, and real team impact.

🎬 Video Demo — 12-minute walkthrough embedded with responsive player

🎭 Interactive Animations — Scroll reveals, typing effect, counter animations

♿ Accessibility — Semantic HTML, ARIA labels, and keyboard navigation

✉️ Contact Form — EmailJS integration for direct email submissions
